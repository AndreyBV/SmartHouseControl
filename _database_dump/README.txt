Версия PostgreSQL: 
	PostgreSQL 10.6, compiled by Visual C++ build 1800, 64-bit
	
Консольная команда для восстановленя дампа: 
	psql -U пользователь -p порт база_данных < файл_дамп.sql 
	где "пользователь" 	- пользователь PostgreSQL обладающий правами на восстановление дамба БД
		"порт" 			- порт на котором работает PostgreSQL (по умполчанию: 5432)
		"база_данных" 	- название пустой базы данных созданной в PostgreSQL
		"файл_дамп"		- путь к файлу с дампом БД	
	Пример:
	psql -U postgres -p 5432 smart_house < d:\smart_house.sql

Строка подключения программного обеспечения для стенда "Умное здание" к базе данных находиться в папке с ПО в файле SettingConnection.json

ПРИМЕЧАНИЕ: 
Для корректной работы консольных команд PostgreSQL необходимо определить переменные среды Windows, порядок действий: 
ПКМ "Мой комьпютер" -> Дополнительные параметры системы -> Переменные среды... -> в пременную "Path" через точку с запятой добавить строку: "c:\Program Files\PostgreSQL\10\bin\"
